					AWS EC2 Services
					################

-> EC2 is the most demanded services  in AWS cloud

-> EC2 stands for Elastic Compute Cloud

-> EC2 is used to create Virtual Machines

	Ex: Linux VM, Windows VM, MAC VM etc....

-> EC2 Virtual machine is called as EC2 instance /VM / Server

EC2 instance billing will be calculated on Hourly basis

-> EC2 instance minimum billing period is 1 hour

Note: With AWS Free tier account 1st one year EC2 instance are free (Monthly 750 hours)

		(t2.micro) 

-> EC2 instance are resizable (Scale up, Scale down)


1) EC2 service is used to create Virtual machines

2) AMI (Amazon machine Image - configuration required for instance)

3) Instance Type (t2.micro - free tier eligible - 1 year - monthly 750 hrs)

4) Key Pair (public & private keys)

5) VPC (Isolated network) - default vpc

6) Security Group (inbound rules & outbound rules)

7) EBS Volume (Elastic Block Storage - 8GB for Linux)

  -> Note: one Key-Pair we can use for multiple EC2 Instances

  -> Note: One Security Group we can use for Mutiple EC2 instance

8) Types of IP's ()Public ip, private ip and elastic ip

					#######
					AWS IAM Services
					#######

=> Identity and Access management

=> It is used to create users, user groups and user permissions

=> When we sign up in AWS, by default we will get Root Account.

=> Root account is very powerfull account, we can access everything in aws using 
   root account.

Note: For team members, root account credentials will not be shared.

=> For every team member, IAM account will be created.

=> For daily activities in AWS  cloud, we will use IAM account only.


=> AS root account is powerful account, we need to provide high security for Root account

=> To provide high security for root account we can use MFA 
	(Multi Factor Authentication)

=> IAM Work by.....
	1) Authentucation: verifing who are 
	2) Authorization: IAM checks your policies to deside what actions you allowed to perform
	3) Access Evaluation Logic: AWS denies by default; only explicit Allow grants access

Note: WE can use Googal Authenticatior app app to provide MFA for root account.
