				==============
			 ###	Linux Commands    ###
				==============

Whoami : Displays logged in user interface

date   : prints todas's date

cal    : print current month calender

pwd    : display present working directory

mkdir  : Create new  directory

rmdir  : Remove empty directory

cd     :    change directory   
	    	-> cd .. : come back directory

Is     :    list files 
		-> ls -l  : list the file in alphabetic order
		-> ls -lr : reverse of alphabetic order
		-> ls-ltr : 
		-> ls -lt : list the file time based 

touch  : To create empty file

cat    : Create file with content & display file content ( create / update / display )

	=>	cat > filename  : Create file with date  (we will loose old data)
	=>	cat >> filename : update date at the bottem
	=>	cat filename 	: display file content

	Note :  cat -> read from top to bottom
		tac -> read bottom to top

rm 	: to remove the file
   rm -r dirname : to delete the non empty  file ( remove recursiv )

mv 	: To rename  file / directory

cp 	: To copy data from one file to another file

wc 	: word count  (line, words, characters)

head    : Give first 10 lines of file (10 is defaolt)

	=>	haed -n 12 <filename>   (gives first 20 lines)

tail 	: Give last 10 lines of file

	=>	tail -n 12 <filename>   (gives last 12 lines )

	=>      tail -f <filename>      (on-growing data)

grap    : global regural expression printer ( it for data finding )

	  =>  grap 'Linux' <filename> : find the operation ( it is case sensitive )

	  =>  grap -i 'as' <filename> : not case sensitive


for editing the file  ( vi <filename> )
vi     :  visual editor ( to modified file content )
	=>  vi <filename>

	    ->	insert made : i
	    ->	save and quite : ESC + :wq
	    ->	quite without saving : ESC and : q!

    	To check it
	   =>  cat <filename>

chmod    : To change file permissions

chown    : To change file ownership
	
check ver: java -version 

Install  : sudo yum install java     

ctrl+t	 : close the file

ctrl+r   : reconnect the VM
